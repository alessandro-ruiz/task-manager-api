# Task Manager API

Backend API for task management built to showcase backend development skills, clean structure, SQL integration, and Docker support.

## Features
- Create, read, update, and delete tasks
- Task status management
- REST API endpoints
- SQL Server integration
- Docker support
- Clean and scalable project structure

## Tech Stack
- C#
- ASP.NET Core
- SQL Server
- Entity Framework Core
- Docker

## Project Goal
This project was created to showcase backend development practices, API design, database integration, and professional project organization.
